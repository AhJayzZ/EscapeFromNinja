## 環境設定 Environment
- 1.開發環境:**Python 3.7.8**
- 2.在終端機執行 ```pip install -r requirements.txt``` 安裝會使用到的套件
- 3.使用工具:
    - 打開角色圖像: ```opencv_contrib_python==4.5.3.56```
    - 撥放音效:```playsound==1.3.0```
    - GUI設計:```PyQt5==5.15.6```
    - 撥放背景音樂:```python_vlc==3.0.12118```
